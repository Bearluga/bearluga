利用“物料绩效管理”创造价值

数字工厂系统可以为每一个完成的订单进行资源消耗统计分析，精确到每一个单件物料。企业管理者得到的数据将是每一道工序的各种物料消耗和产出实际数据，可以分别与本订单计划数值以及年度规划数值进行比对，对差异进行分析和整改。 我们采用的是直接成本责任体系，因此任何物料的绩效成果都能与一个具体的岗位负责人对应起来。通过当事人的行为，最终逐步改善各种物料的投入产出关系，实现物尽其用。

利用“人员绩效管理”创造价值

数字工厂系统可以为企业的各个工种设计独特的绩效管理分析报表。员工每完成一项工作，系统会自动记录工作完成情况以及绩效分析，每个人可以动态掌握本职工作各个方面的实时成果，无需他人提醒。这相当于为每一个岗位配置了一台数字摄像头，员工工作绩效全部根据客观的数据依据，主观的、形式主义的评判不再是主流。 在数字工厂系统中，人员绩效管理最终能够与岗位的责权利一一对应，形成一种鼓励个人和团队积极向上的工作机制，实现人尽其能。

利用“设备绩效管理”创造价值

作为企业重要成本的机器设备来说，设备的投入产出比是关系到企业产品成本重要因素。数字工厂系统中，可以为每一台设备设计独特的绩效管理分析报表。设备绩效管理将对设备的使用效率、产出效率、合格品率、维护效率等方面提供分析报告，为企业管理者深度挖掘企业成本及生产率提供客观数据支撑，是企业高级管理的重要工具。
